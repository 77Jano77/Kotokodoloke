# Carpeta de Banners Publicitarios

Esta carpeta contiene los banners publicitarios que se mostrarán en el lateral izquierdo de la página.

## Cómo agregar banners:

1. Coloca tus imágenes de banner en esta carpeta (`public/publi/`)
2. Los formatos aceptados son: `.png`, `.jpg`, `.jpeg`, `.gif`, `.webp`
3. Tamaño recomendado: **300x250px** o **300x600px** (dimensiones estándar de banners)
4. Nombre archivos como: `banner1.png`, `banner2.png`, etc.

## Gestión de banners:

- El administrador puede seleccionar qué banners mostrar desde el panel de gestión (botón "⚙️ Gestionar Banners")
- Solo los banners seleccionados aparecerán visibles en la página
- Los banners se muestran en el orden en que fueron agregados

## Ejemplos de nombres de archivo:

- `banner1.png` - Banner principal
- `banner2.jpg` - Banner secundario
- `promo-verano.png` - Promoción de verano
- `sponsor-pokemon.gif` - Banner animado de sponsor

**Nota:** Los usuarios normales verán solo los banners que el admin haya activado.
