# Recursos del Torneo

Esta carpeta contiene recursos multimedia como videos, GIFs e imágenes para el torneo.

## Archivos requeridos:

### Video de Intro:
- **video intro.mp4** - Video de introducción para la página de INICIO

### GIF de Ruleta:
- **delibird.gif** - GIF animado que se muestra durante el sorteo en la RULETA

### Medallas de Kanto (Medallero):
Las 8 medallas en orden:
1. **Roca.png** - Medalla Roca (Boulder Badge)
2. **Cascada.png** - Medalla Cascada (Cascade Badge)
3. **Trueno.png** - Medalla Trueno (Thunder Badge)
4. **Arcoiris.png** - Medalla Arcoíris (Rainbow Badge)
5. **Alma.png** - Medalla Alma (Soul Badge)
6. **Pantano.png** - Medalla Pantano (Marsh Badge)
7. **Volcan.png** - Medalla Volcán (Volcano Badge)
8. **Tierra.png** - Medalla Tierra (Earth Badge)

## Uso:

### Video de INICIO:
Ruta por defecto: `/recursos/video intro.mp4`

### GIF de RULETA:
Ruta por defecto: `/recursos/delibird.gif`

### Medallas:
Se muestran automáticamente en las fichas de jugadores usando:
```
/recursos/Roca.png
/recursos/Cascada.png
etc.
```

## Formatos soportados:

- **Video**: MP4, WEBM, OGG
- **Imágenes**: PNG, JPG, GIF
- **Músicas**: Van en la carpeta `/music/`
